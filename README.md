# 📰 The Daily Ledger

A clean and elegant newspaper-inspired To-Do application built using **HTML, CSS, and JavaScript**.

## ✨ Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 🔍 Filter tasks (All / Open / Done)
- 📊 Live task counters
- 🕒 Timestamp for every task
- 📱 Responsive design
- ♿ Keyboard accessible

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Future Improvements

- Dark Mode
- Drag & Drop Tasks
- Categories
- Due Dates
- Search Functionality
