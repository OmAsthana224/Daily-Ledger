# 📰 The Daily Ledger

A clean and elegant newspaper-inspired To-Do application built using **HTML, CSS, and JavaScript**.

## ✨ Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 🔍 Filter tasks (All / Open / Done)
- 📊 Live task counters
- 🕒 Timestamp for every task
- 📱 Responsive design
- ♿ Keyboard accessible

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## 📂 Project Structure

```
The-Daily-Ledger/
│
├── index.html

├── script.js

```

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/yourusername/the-daily-ledger.git
```

2. Open the project folder.

3. Run `index.html` in your browser.

No installation required.
